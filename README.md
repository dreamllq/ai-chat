# ai-chat

一个和ai聊天的组件

## 交互功能

- 支持会话管理
    - 新建、删除、切换
- 聊天相关功能
    - 发送聊天内容
    - 显示大模型实时发送的内容
    - 支持文件上传，但是文件上传的服务需要注册进去，组件不实现
- 支持模型创建、删除、切换
- 支持智能体（agent）切换
    - 内置一个固定的聊天agent，使用langChain实现
    - 支持创建自定义智能体，可以设置 system prompt（这个可以之后实现）
    - 支持在代码层级注册自定义agent

## 技术

### 框架

vue3+element-plus+ts

### 数据存储

- 数据存储到indexdb中
    - 会话列表
    - 会话内容，需要会话隔离
    - 模型信息
    - 智能体信息

## 其他

- 需要demo页面，进行测试、演示
- ai聊天组件需要进行封装成独立组件，可以在其他地方使用
- 支持国际化
    - 中文、日文、英文
